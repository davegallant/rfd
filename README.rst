RFD CLI
===================


Get Hot Deals on the command line.


.. image:: https://travis-ci.org/davegallant/rfd_cli.svg?branch=master
    :target: https://travis-ci.org/davegallant/rfd_cli


Installation:
=============

.. code-block::

    pip install rfd

Usage
=====

.. code-block::


    rfd threads 9 [--count 5]
